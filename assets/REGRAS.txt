## 3. Regras Técnicas de Output (HTML/Blogger)

Você gerará um **FRAGMENTO DE HTML** para o corpo do post.

### Proibições Estruturais

1. **NUNCA use:** `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`.
2. **NUNCA use tags:** `<meta>` ou `<title>` (cabeçalho do Blogger).
3. **NUNCA use:** `<h1>` no corpo do texto (apenas `<h2>`, `<h3>`, etc.).

### Regras de Tabela (Mobile-First)

Se usar tabelas, aplique este CSS Inline RIGOROSAMENTE:

* **Wrapper:** `<div style="overflow-x: auto; width: 100%; margin-bottom: 20px;">`
* **Table:** `<table style="width: 100%; min-width: 600px; border-collapse: collapse;">`
* **TH/TD:** `style="padding: 12px; border: 1px solid #cccccc; word-break: keep-all; hyphens: none;"`

### Protocolo Anti-Alucinação (Local SEO)

1. **Geografia Real:** Apenas bairros/pontos reais de Indaiatuba.
2. **Distâncias:** Use tempos realistas (ex: "A 12 min de Viracopos").
3. **Honestidade:** Não invente depoimentos. Se não tiver dados, generalize com lógica (ex: "escolas na região").
