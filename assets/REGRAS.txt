### REGRAS TÉCNICAS DE OUTPUT (HTML/BLOGGER)

Você gerará um FRAGMENTO DE HTML para o corpo de um post.

PROIBIÇÕES ESTRUTURAIS:
I. NUNCA use: <!DOCTYPE html>, <html>, <head>, <body>.
II. NUNCA use tags <meta> ou <title> (isso vai no cabeçalho do Blogger).
III. NUNCA use <h1> dentro do corpo do texto (use apenas <h2>, <h3>, etc.).

REGRAS DE TABELA (MOBILE-FIRST):
Se usar tabelas, aplique este CSS Inline RIGOROSAMENTE para evitar quebra no mobile:
1. Wrapper: <div style="overflow-x: auto; width: 100%; margin-bottom: 20px;">
2. Table: <table style="width: 100%; min-width: 600px; border-collapse: collapse;">
3. TH/TD: style="padding: 12px; border: 1px solid #cccccc; word-break: keep-all; hyphens: none;"

PROTOCOLO ANTI-ALUCINAÇÃO (LOCAL SEO):
1. **Geografia Real:** Use apenas bairros e pontos de referência REAIS de Indaiatuba.
2. **Distâncias:** Use tempos realistas (ex: "A 12 min de Viracopos").
3. **Honestidade:** Não invente depoimentos. Se não tiver um dado específico, generalize com lógica (ex: "escolas na região") em vez de inventar um nome.
